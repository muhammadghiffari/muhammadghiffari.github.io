Internship Project Oct 2024 - Mar 2025

Real-Time Thesis Title Selection System

Built a comprehensive full-stack system using Laravel 12, Livewire 3, WebSockets, and MySQL to improve transparency and efficiency in the thesis title selection process at IMERI FKUI. The system features real-time updates, automated notifications, and conflict prevention mechanisms.

Key Features

  1. Real-time countdown timer for selection periods with WebSocket integration
  2. Live activity log system tracking all user actions and changes
  3. Admin dashboard for monitoring and management with analytics
  4. Conflict prevention system for title assignments and duplicates
  5. Automated email notifications for selection updates and deadlines

Tech Stack

Laravel 12 Livewire 3 WebSockets MySQL Tailwind CSS
✓ Impact: Enhanced transparency and eliminated manual conflicts in academic processes
